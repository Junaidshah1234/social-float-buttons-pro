## 🚀 Installation

1. Download ZIP from releases
2. WordPress Admin → Plugins → Add New → Upload Plugin
3. Activate
4. **Social Float** menu → Settings
5. Configure button text, color, and enable platforms
6. Save Changes — done!

---

## ⚙️ How to Use

1. Go to **Social Float** in WordPress admin
2. Set your button text (e.g., "📱 Contact Me")
3. Pick a button color
4. Enable the platforms you want
5. Fill in your profile URLs
6. For custom icons — paste SVG code from simpleicons.org
7. Save and check your site!
