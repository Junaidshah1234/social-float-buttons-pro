Floating social media contact buttons with toggle popup. WhatsApp, LinkedIn, GitHub, Upwork, Instagram, Twitter, YouTube & more.

== Description ==

Social Float Buttons Pro adds a beautiful floating contact button to your WordPress site. Click the button and all enabled social media icons pop up with smooth animation.

**Features:**

- Toggle popup with smooth animation
- 11 social platforms: WhatsApp, LinkedIn, GitHub, Upwork, Instagram, Twitter/X, YouTube, Telegram, Discord, Email, Phone
- Customizable button text & color
- WhatsApp pre-filled message support
- Left or right position
- Enable/disable individual icons
- Clean, modern admin settings page
- Mobile responsive
- Security hardened (nonces, escaping, sanitization)
- No external dependencies
- Translation ready

**Use Cases:**

- Freelancer portfolio sites
- Business websites
- Personal blogs
- Agency websites
- E-commerce stores

== Installation ==

1. Upload the `social-float-buttons-pro` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu
3. Go to **Social Float** menu in admin sidebar
4. Configure button text, color, and position
5. Enable desired social platforms and fill URLs
6. Save Changes

== Frequently Asked Questions ==

= Can I change the button color? =

Yes! Use the color picker in settings or enter any hex color code.

= Does WhatsApp work on desktop? =

Yes! It opens WhatsApp Web on desktop and WhatsApp app on mobile.

= Is it mobile responsive? =

Absolutely! The buttons scale perfectly on all devices.

= Can I reorder the icons? =

Icon order follows the settings page order. Enable/disable to control which show.

= Is it secure? =

Yes! Built with WordPress security best practices - nonces, escaping, sanitization, capability checks.
