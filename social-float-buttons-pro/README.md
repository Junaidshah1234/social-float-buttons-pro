## 🚀 Installation

1. Download ZIP
2. WordPress → Plugins → Add New → Upload
3. Activate
4. Social Float → Settings
5. Configure & Save

## ⚙️ Settings

- Button text (emojis supported)
- Button color (color picker or hex)
- Position (bottom right/left)
- Enable/disable each platform
- Custom URLs & WhatsApp message
